- Real-time video analysis using **OpenCV + YOLOv8 tracking**  
- Evidence capture with **violation type, date, and time**  
- Interactive **Flask web interface** for video upload and violation monitoring 

### 🎓 AI Career Advisor Chatbot

- **Advanced Prompt Engineering** for structured AI responses  
- **Multi-turn conversation memory** for contextual recommendations  
- Interactive **Streamlit chat interface** for user-friendly experience  
- **Safety guardrails** to restrict non-career queries
